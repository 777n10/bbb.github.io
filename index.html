關於我

我是伍妡妮，出生於民國94年5月10日，
現在就讀於靜宜大學資料科學暨大數據分析與應用學系。
在家中我是排行老二，有一個哥哥與弟弟，哥哥是東海大學的大四生，而弟弟還是高中生， 我們年齡相近，相處自然也都很和睦，是個和樂的家庭。也因為在這種家庭成長的關係， 我對於事情都抱持著樂觀，即使遇到失敗也不氣餒，只要肯努力就會越來越靠近成功的那天。

以大橘為重

我家有兩隻橘貓，我平常的興趣就是陪牠們玩，牠們都是因緣巧合來到我們家。
其中一隻叫哇哇，因為走錯路來到一座山上，只記得那天特別冷，在這杳無人煙的地方， 腳邊居然竄出一隻小貓，牠渾身發抖的模樣讓人心生憐憫，牠一路跟著我們，我們也就把牠帶了回家。
另一隻則是叫妞妞，牠曾經有過主人，但卻流落到了街頭，在外頭沒能吃好睡好，讓牠遍體鱗傷滿身重病，在被愛媽找到後，被發上網路找新主人，我們家自告奮勇的要養牠，不料當牠被送來時卻與照片截然不同， 也是呢誰會想領養一隻生重病的貓，但我們還是接受了牠的到來，並且相信著牠一定會好起來的，在跑了好幾趟醫院與對牠一日日的照顧下，奇蹟發生似的完全康復了，也許愛也是一種解藥。

import requests
from bs4 import BeautifulSoup

def get_drink_description():
url = "https://www.8teahouse.com/article_d.php?lang=tw&tb=2&id=83"
headers = {
"User-Agent": "Mozilla/5.0"}
response = requests.get(url, headers=headers)
response.encoding = 'utf-8'
soup = BeautifulSoup(response.text, 'html.parser')

article_content = soup.find('div', class_='drinks_ptxt')
if article_content:
titles = article_content.find_all('h3')
target_name = "綠豆沙珍珠厚奶"
description = ""

for title in titles:
if target_name in title.text:
next_p = title.find_next_sibling('p')
if next_p:
description = next_p.text.strip()
break

if description:
print(f"{target_name} 說明：\n{description}")
else:
print(f"找不到「{target_name}」的說明")
else:
print("找不到文章內容區塊")

if __name__ == "__main__":
get_drink_description()
